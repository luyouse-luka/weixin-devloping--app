# 字体下载和配置完整指南

## 推荐字体：站酷快乐体（可爱风格）

最适合家庭点餐小程序的可爱风格字体。

## 下载步骤

### 方法一：从站酷官网下载（推荐）

1. **访问站酷字体官网**
   - 网址：https://www.zcool.com.cn/special/zcoolfonts/
   - 或者搜索"站酷字体"进入官网

2. **下载站酷快乐体**
   - 在字体列表中找到"站酷快乐体"
   - 点击下载按钮
   - 选择"免费下载"或"商用免费"

3. **解压文件**
   - 下载后解压zip文件
   - 找到 `.ttf` 格式的字体文件（通常是 `站酷快乐体.ttf`）

4. **放置字体文件**
   - 将 `.ttf` 文件复制到项目的 `fonts` 文件夹
   - 重命名为 `cute-font.ttf`（方便使用）

### 方法二：备用下载地址

如果站酷官网无法访问，可以尝试：

1. **GitHub字体库**
   - 搜索 "zcool fonts github"
   - 查找字体资源

2. **字体网站**
   - 字客网：https://www.fontke.com/
   - 字体天下：http://www.fonts.net.cn/
   - 搜索"站酷快乐体"下载
